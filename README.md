# 아대숲 온라인 알고리즘 스터디 관리

이 프로젝트는 아대숲 온라인 알고리즘 스터디에서 결과물을 일일 단위로 업로드했는지 안 했는지를 체크해 주는 웹 사이트 프로젝트입니다.

## 기술 스택 소개

- Node 12.18.2
- Yarn 1.22.4
- React 16.9.0
- TypeScript ~3.7.2
- GitHub Actions
